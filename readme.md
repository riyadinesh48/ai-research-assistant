# AI Research Assistant

## What it does
This app takes any topic from the user and automatically:
- Searches the web for information
- Uses AI to read and summarize results
- Generates a full research report

## How to Run
1. Install dependencies: pip install anthropic tavily-python streamlit python-dotenv groq
2. Add your API keys in .env file
3. Run: streamlit run app.py

## Tech Stack
- Python
- Streamlit for UI
- Tavily API for web search
- Groq AI for report generation

## Limitations
- Needs internet connection
- Free API limits apply
